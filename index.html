Після розархівування відкривати:

  • index.html - це головна
  • services.html - це сторінка з послугами
  • about.html - це сторінка про компанію
  • services.html - це сторінка з контактами

Папки:

  • css та sass - папки з таблицями стилів css
  • fonts - це папка з шрифтами
  • images - це папка з зображеннями
  • js - скрипти
  • favicon.ico - іконка для сайту
